## 1. 기획 의도

- 집중(Focus/Stopwatch)과 퀘스트 완료라는 **생산성 행동**을 골드 보상으로 연결하여 게이미피케이션 루프 구축
- 골드는 길드원 모집 등 수집형 콘텐츠의 **소모처**와 연결되어 "집중 → 보상 → 수집" 순환을 형성
- 획득·소모 규칙을 한 곳에서 관리하여, Focus·Quests·Guild 등 여러 시스템이 일관된 경제 규칙을 참조

## 2. 아이디어

### 2.1. 기본 방향

골드는 RecordTales의 **단일 범용 재화**이다. 집중 세션(타이머/스톱워치) 가동과 퀘스트 완료로 획득하고, 길드원 모집 등 게이미피케이션 콘텐츠에 소모한다.

### 2.2. 골드 획득 — 집중 세션

| 항목 | 규칙 |
|------|------|
| 적립 주기 | **6초마다** |
| 적립량 | **10골드** |
| 10분 집중 환산 | 1,000골드 (= 길드원 모집 1회분) |
| 적용 대상 | 타이머 가동 중 · 스톱워치 가동 중 |
| 능력치 보너스 | 길드원 능력치로 획득량 증가 가능 (+n 또는 +%) |

- 타이머/스톱워치가 **가동 중**일 때만 적립
- 일시정지·종료 시 적립 중단

### 2.3. 골드 획득 — 퀘스트 완료

#### 2.3.1. 설정 방식

- 퀘스트 생성/수정 시 사용자가 **직접 보상 골드를 지정**
- 입력 단위: **100골드 단위** (자유 입력, 끝 자릿수 0 유지)
- 프리셋 칩: `100` · `300` · `500` · `1,000` 골드 중 선택 가능

#### 2.3.2. 일일 상한

- 퀘스트 완료로 획득 가능한 골드는 **길드 레벨에 따라** 일일 상한이 변동
- 상한 초과 시 추가 골드 미지급
- **자정(00:00) 기준 리셋**

| 길드 레벨 | 일일 상한 | 해금 키 |
|-----------|----------|---------|
| Lv.1~3 | 2,000G | (기본) |
| Lv.4~6 | 2,500G | `goldcap2500` (자동 해금) |
| Lv.7+ | 3,000G | `goldcap3000` (자동 해금) |

### 2.4. 골드 소모처

| 소모처 | 비용 | 비고 |
|--------|------|------|
| 길드원 모집 | 1,000골드 / 1회 | `guild-member-recruit` 참조 |

#### 2.4.1. 해금 상점 구매처 (unlock-shop)

| 해금 항목 | 비용 | 해금 조건 |
|-----------|------|-----------|
| 스쿼드 프리셋 +1 (1차) | 3,000G | 길드 Lv.4 |
| 스쿼드 프리셋 +1 (2차) | 5,000G | 길드 Lv.8 |
| 편성 슬롯 +1 (1차) | 5,000G | 길드 Lv.5 |
| 편성 슬롯 +1 (2차) | 8,000G | 길드 Lv.9 |
| 앰비언트 사운드: Thunder ⚡ | 1,500G | 길드 Lv.2 |
| 앰비언트 사운드: Cave 🦇 | 2,000G | 길드 Lv.5 |
| 앰비언트 사운드: Tavern 🍺 | 3,000G | 길드 Lv.8 |

- 향후 추가 소모처 확장 가능 (장비, 꾸미기 등)

### 3.3. 퀘스트 보상 골드 설정 UI

```
┌─ 퀘스트 보상 골드 ─────────────────────────┐
│                                             │
│  [100] [300] [500] [1,000]  ← 프리셋 칩     │
│                                             │
│  ┌──────────────────────┐                   │
│  │ 직접 입력 (100 단위)  │  골드             │
│  └──────────────────────┘                   │
│                                             │
│  오늘 남은 상한: 1,400 / 2,000              │
└─────────────────────────────────────────────┘
```

- 퀘스트 생성/수정 폼에 추가되는 필드
- 프리셋 칩 선택 시 입력 필드에 자동 반영
- 하단에 오늘 남은 일일 상한 표시
