## 1. 기획 의도

- 집중(Focus/Stopwatch)과 퀘스트 완료라는 **생산성 행동**을 골드 보상으로 연결하여 게이미피케이션 루프 구축
- 골드는 길드원 모집, 기능 해금, 콘텐츠 해금, 레벨업 등 다양한 **소모처**와 연결되어 "집중 → 보상 → 수집/성장" 순환을 형성
- 획득·소모 규칙을 한 곳에서 관리하여, Focus·Quests·Guild 등 여러 시스템이 일관된 경제 규칙을 참조

## 2. 아이디어

### 2.1. 기본 방향

골드는 RecordTales의 **단일 범용 재화**이다. 집중 세션(타이머/스톱워치) 가동과 퀘스트 완료로 획득하고, 길드원 모집·기능 해금·콘텐츠 해금·레벨업에 소모한다.

### 2.2. 골드 획득 — 집중 세션

| 항목 | 규칙 |
|------|------|
| 적립 주기 | **6초마다** |
| 적립량 | **10골드** |
| 10분 집중 환산 | 1,000골드 (= 길드원 모집 1회분) |
| 적용 대상 | 타이머 가동 중 · 스톱워치 가동 중 |
| 능력치 보너스 | 길드원 능력치로 획득량 증가 가능 (+n 또는 +%) |

- 타이머/스톱워치가 **가동 중**일 때만 적립
- 일시정지·종료 시 적립 중단

### 2.3. 골드 획득 — 퀘스트 완료

#### 2.3.1. 설정 방식

- 퀘스트 생성/수정 시 사용자가 **직접 보상 골드를 지정**
- 입력 단위: **100골드 단위** (자유 입력, 끝 자릿수 0 유지)
- 프리셋 칩: `100` · `300` · `500` · `1,000` 골드 중 선택 가능

#### 2.3.2. 일일 상한

- 퀘스트 완료로 획득 가능한 골드는 **일일 2,000골드 상한** (고정, `questGoldDailyCap: 2000`)
- 상한 초과 시 추가 골드 미지급
- **자정(00:00) 기준 리셋** (`questGoldToday` 초기화)
- **구현 상태**: 인프라(상한 변수·리셋 로직) 구현 완료, 퀘스트 완료 시 골드 지급 로직은 미연결 (현재 `toggleInstanceDone`은 XP만 지급)

### 2.4. 골드 소모처

| 소모처 | 비용 | 비고 |
|--------|------|------|
| 길드원 모집 | 1,000골드 / 1회 | `HIRE_COST = 1000` |
| 레벨업 | `500 × level` 골드 | `tryGuildLevelUp()` — XP + 골드 충족 시 수동 클릭 |

#### 2.4.1. 기능 해금 상점 (UNLOCKS_V2)

### 2.5. 보상 배수 시스템

기능 해금(UNLOCKS_V2)으로 보상 배수가 카테고리별로 증가 (`getRewardMultiplier` 함수):

| 카테고리 | 최소 | 최대 | 단계당 증가 |
|----------|------|------|-------------|
| 스톱워치 보상 | ×1.0 | ×2.0 | +20% per tier |
| 타이머 보상 | ×1.0 | ×2.0 | +20% per tier |
| 퀘스트 보상 | ×1.0 | ×2.0 | +20% per tier |

### 2.6. 초기 자산

| 항목 | 값 | 코드 변수 |
|------|-----|----------|
| 시작 골드 | 5,000골드 | `A.gold = 5000` |

> 모집 5회분. 신규 유저가 즉시 수집을 경험할 수 있도록 설정.

## 3. 예상 UI (대략적)

### 3.1. 골드 잔액 표시

- `guild-info` (Guild Plate)에 `💰 ${A.gold.toLocaleString()}` 형태로 표시

### 3.2. 집중 세션 중 골드 적립 표시

- `goldTick()` 함수로 6초마다 적립

### 3.3. 퀘스트 보상 골드 설정 UI

```
┌─ 퀘스트 보상 골드 ─────────────────────────┐
│                                             │
│  [100] [300] [500] [1,000]  ← 프리셋 칩     │
│                                             │
│  ┌──────────────────────┐                   │
│  │ 직접 입력 (100 단위)  │  골드             │
│  └──────────────────────┘                   │
│                                             │
│  오늘 남은 상한: 1,400 / 2,000              │
└─────────────────────────────────────────────┘
```

## 6. 관련 시스템

| 시스템 | 관계 |
|--------|------|
| `focus-panel` | 타이머/스톱워치 가동 중 6초마다 10골드 적립 |
