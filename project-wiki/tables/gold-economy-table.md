# 골드 경제 테이블 (Gold Economy Table) — v2

> **문서 성격**: 골드 획득·소모 관련 수치 데이터 시트.
> 코드(`RecordTales.html`)의 경제 상수와 1:1 대응.

---

## 1. 골드 획득

### 1.1. 집중 세션 적립

| 항목 | 값 | 코드 변수 |
|------|-----|----------|
| 적립 주기 | 6초 | `goldTick()` 내 카운터 6 |
| 적립량 | 10골드 | `A.gold += 10` |
| 10분 환산 | 1,000골드 | — |
| 30분 환산 | 3,000골드 | — |
| 1시간 환산 | 6,000골드 | — |

> 타이머/스톱워치 **가동 중**에만 적립. 일시정지·휴식 중에는 적립 안 됨.

### 1.2. 퀘스트 완료 보상

| 항목 | 값 | 비고 |
|------|-----|------|
| 보상 단위 | 100골드 단위 | 자유 입력 |
| 프리셋 칩 | 100 / 300 / 500 / 1,000 | 빠른 선택용 |
| **일일 상한** | **2,000골드** | 자정(00:00) 리셋 |
