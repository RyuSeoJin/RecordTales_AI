## 5. 레벨 곡선 설계 원칙

| 원칙 | 설명 |
|------|------|
| **초기 진입 장벽 낮게** | Lv.1 → Lv.2는 50 XP (1~2일) |
| **지수 곡선** | 이후 레벨은 점진적으로 필요 XP 증가 |
| **일일 플레이 기준** | 하루 1~2시간 집중 시 40~60 XP |
| **보상 체감 방지** | 레벨업마다 의미 있는 해금 요소 제공 |
